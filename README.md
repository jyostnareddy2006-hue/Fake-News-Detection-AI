#  Fake News Detection AI

A simple Fake News Detection system built using Python.

## Project Description
This project detects whether a news article is Fake or Real 
based on predefined suspicious keywords.

It is a beginner-friendly implementation of a text classification concept.

---

##  How It Works

1. User enters a news sentence.
2. The system checks for fake-related keywords.
3. If suspicious words are found → Output: Fake News
4. Otherwise → Output: Real News

---

##  Project Structure

- app.py → Main program file
- model.py → Contains prediction logic
- README.md → Project documentation

---

##  How to Run

1. Install Python
2. Download the project
3. Open terminal
4. Run:
   python app.py
